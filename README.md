# DSA Repository

Welcome to the **Data Structures & Algorithms (DSA) Repository**!
This repository is organized to help you systematically learn DSA, implement solutions, and prepare for interviews and competitive programming.

---

## Folder Structure

```

  00\_prompt
  01\_Fundamentals
  02\_Data\_Structures
  03\_Algorithms
  04\_Problem\_Patterns

```

---

### 00_prompt
Contains **prompts, instructions, and notes** for using this repository.
- Example contents:
  - How to navigate the repo
  - Coding standards and guidelines
  - Best practices for DSA study

---

### 01_Fundamentals
Covers **core concepts and theoretical foundations**.
- Topics include:
  - Time & space complexity
  - Big-O, Big-Theta, Big-Omega
  - Recursion fundamentals
  - Number theory basics (gcd, lcm, primes)
  - Math tricks and bit manipulation

---

### 02_Data_Structures
All major **data structures**, their theory, implementations, and analyses.
- Subfolders:
  - `arrays/` → Array implementations, analyses, and problem sets
  - `linked_lists/` → Singly, doubly, and circular linked lists
  - `stacks_queues/` → Stack, Queue, Priority Queue, Deque
  - `trees/` → Binary trees, BST, Segment trees, Trie
  - `graphs/` → Graph representations, traversals, algorithms
  - `hashing/` → Hash maps, sets, collision handling
  - `heaps/` → Min-heap, max-heap, heap operations
- Each structure includes:
  - Theory, pseudocode, and optimized code
  - Time-space complexity analysis
  - Interview tips
  - Practice problems

---

### 03_Algorithms
Contains **algorithmic paradigms and classic problems**.
- Subfolders:
  - `sorting/` → Quick sort, Merge sort, Heap sort, etc.
  - `searching/` → Binary search, Ternary search
  - `dynamic_programming/` → Memoization, Tabulation, Classic DP problems
  - `greedy/` → Greedy algorithms with examples
  - `graph_algorithms/` → BFS, DFS, Dijkstra, MST
  - `divide_and_conquer/` → Key D&C problems
  - `backtracking/` → N-Queens, Sudoku, Subset generation

- Each algorithm contains:
  - Problem description
  - Code implementations (Python/C++)
  - Complexity analysis
  - Variations & edge cases

---

### 04_Problem_Patterns
Organizes problems by **common patterns and strategies**.
- Example patterns:
  - Sliding window
  - Two pointers
  - Fast & slow pointers
  - Recursion & backtracking patterns
  - Graph traversal patterns
  - Greedy & DP patterns

- Helps in **interview preparation** and **pattern recognition**.

---

## Recommended Usage

1. Start with `01_Fundamentals` to build strong foundations.
2. Explore `02_Data_Structures` and implement each structure.
3. Move to `03_Algorithms` for algorithmic paradigms.
4. Solve problems in `04_Problem_Patterns` to identify recurring strategies.
